# Rapid - Browser Dashboard
A modern dashboard extension that replaces the regular browser home-page with an informative, elegant and customizable dashboard. This extension works best on Chrome since the Identity API is not supported by other browsers.

## Current Features
- **Search Bar:** allows you to search in Google
- **Custom Cursor:** custom circular cursor with interactive animations to replace the old, normal and boring cursor.
- **Bookmarks (Not Complete):** clickable and rearrangeable bookmarks; I'm working on allowing users to add / remove bookmarks
- **Custom Context Menu (Not Complete):** a custom context menu that currently allows you to:
    - Switch between light and dark mode; these settings are saved forever in your browser
    - Customize the accent color (neon orange by default) with RGB sliders

## Widgets
- **Einthusan / Movie Widget:** shows a gallery view of the 7 latest Tamil movies on Einthusan, an Indian streaming service
- **Weather Widget:** shows the current weather and temperature for Ottawa, Ontario
- **Clock Widget:** shows the current time, updating every 4 seconds
- **Battery Widget (Not Complete):** shows your device's current battery percentage. I am currently trying to implement the battery percentage of multiple devices through Bluetooth
- **Gmail Widget:** shows your 5 most recent emails that can be clicked to be redirected to that email in the Gmail website
- **Applet Widget: (Not Complete)** shows a collection of websites that you can click to be redirected to
